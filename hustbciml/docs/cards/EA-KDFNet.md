# EA-KDFNet

**Axis:** Network (backbone) &nbsp;·&nbsp; **Paradigm:** MI &nbsp;·&nbsp; **Dataset:** BNCI2014001 (cross-subject LOSO)

**Stage configuration:** `EA · no-aug · KDFNet · Linear · ERM`

## Mechanism
KDFNet (knowledge-data fusion network) mirrors the FBCSP pipeline inside a CNN: a windowed-sinc FIR filter bank supplies fixed, data-free band-pass temporal filters, per-band Common Spatial Pattern filters estimated on the EA-aligned source initialize the spatial layer, and log-variance features feed the head — then the whole network is fine-tuned end-to-end. It fuses classical FBCSP prior knowledge with data-driven training.
