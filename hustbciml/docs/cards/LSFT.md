# LSFT

**Axis:** Classical (network-free) &nbsp;·&nbsp; **Paradigm:** MI &nbsp;·&nbsp; **Dataset:** BNCI2014001 (cross-subject LOSO)

**Stage configuration:** `own RA · tangent-space features · (no backbone) · LDA+LR voting + DJP-MMD adaptation · fit`

## Mechanism
Lightweight Source-Free Transfer keeps no raw source data at transfer time: pretrained source classifiers vote to pseudo-label the target, the confident (low-disagreement) target trials form a virtual intermediate source, and an iterative DJP-MMD subspace adaptation with pseudo-label refinement produces the final target predictions. Everything runs on classical Riemannian tangent-space features; it is transductive (it adapts to the unlabeled target), which distinguishes it from the no-transfer classical baselines.

## Provenance
- **Paper / source:** Zhang & Wu, IEEE Trans. Cognitive and Developmental Systems 2022 (Lightweight Source-Free Transfer, LSFT); github.com/chamwen/LSFT — CA+LSFT (LDA+LR voting) config
- **Implementation:** Adapted from github.com/chamwen/LSFT (Zhang & Wu, IEEE TCDS 2022), the CA+LSFT (shrinkage-LDA + logistic-regression voting) configuration, onto the fit-mode Strategy ABC: fit trains the source voters and discards source raw data (source-free); predict does the voting, virtual-source selection, and iterative feature adaptation on the target. LSFT does its own Riemannian centroid alignment, so the preset aligner is Identity. Minimal disclosed guards cover degenerate virtual-source selection. Requires pyriemann + scikit-learn.
