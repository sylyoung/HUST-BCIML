# PAT

**Axis:** Transfer / adaptation strategy &nbsp;·&nbsp; **Paradigm:** MI &nbsp;·&nbsp; **Dataset:** BNCI2014001 (cross-subject LOSO)

**Stage configuration:** `EA · Scaling · EEGNet · Linear · PAT`

## Mechanism
Privacy-preserving Adversarial Transfer extends ABAT: after a clean warmup, each Euclidean-aligned batch is first amplitude-scaled (multiplied by 1±0.05) and then perturbed by a global-epsilon L∞ projected gradient descent (PGD) attack with a noisy uniform initialization (eps 0.03, alpha 0.005, 10 steps), training on the adversarial examples. The target is never used, so training stays source-only — the sense in which it is privacy-preserving — and the only change from ERM is the augmentation-plus-adversarial inner loop.
