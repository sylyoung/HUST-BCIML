# SAR

**Axis:** Transfer / adaptation strategy &nbsp;·&nbsp; **Paradigm:** MI &nbsp;·&nbsp; **Dataset:** BNCI2014001 (cross-subject LOSO)

**Stage configuration:** `EA · no-aug · EEGNet · Linear · SAR`

## Mechanism
Sharpness-Aware and Reliable test-time adaptation minimizes temperature-scaled prediction entropy over all parameters using a Sharpness-Aware Minimization (SAM) two-step update, so adaptation seeks a flat, stable minimum robust to noisy target batches.

| Metric | Value |
|---|--:|
| Accuracy (mean ± std across seeds) | 74.90 ± 1.99 |
| Δ vs baseline (EA-EEGNet, 72.07) | +2.83 |

Accuracy is the mean over seeds; ± is the standard deviation *across* seeds (a reproducibility figure, not the cross-subject spread). The reference range is the published / expected band on this dataset.
