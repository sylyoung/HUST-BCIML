# MCC

**Axis:** Transfer / adaptation strategy &nbsp;·&nbsp; **Paradigm:** MI &nbsp;·&nbsp; **Dataset:** BNCI2014001 (cross-subject LOSO)

**Stage configuration:** `EA · no-aug · EEGNet · Linear · MCC`

## Mechanism
Minimum Class Confusion adds a loss that minimizes the off-diagonal class-confusion of the temperature-rescaled prediction correlation on the unlabeled target, discouraging ambiguous cross-class predictions. It is not adversarial — a single confusion penalty on the target — and it leads the entire benchmark here.
