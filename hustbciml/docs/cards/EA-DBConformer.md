# EA-DBConformer

**Axis:** Network (backbone) &nbsp;·&nbsp; **Paradigm:** MI &nbsp;·&nbsp; **Dataset:** BNCI2014001 (cross-subject LOSO)

**Stage configuration:** `EA · no-aug · DBConformer · Linear · ERM`

## Mechanism
A dual-branch convolutional transformer. A temporal branch and a spatial branch each tokenize the EA-aligned trial with convolutions, add positional embeddings, and pass the tokens through a multi-head self-attention encoder; channel-attention pooling then fuses the two branches before the linear head. It combines convolutional locality with transformer global context, and leads the backbone axis.
