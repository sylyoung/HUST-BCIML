# EA-TIEEEGNet

**Axis:** Network (backbone) &nbsp;·&nbsp; **Paradigm:** MI &nbsp;·&nbsp; **Dataset:** BNCI2014001 (cross-subject LOSO)

**Stage configuration:** `EA · no-aug · TIEEEGNet · Linear · ERM`

## Mechanism
TIE-EEGNet is EEGNet with its first temporal convolution replaced by a time-information-enhanced (TIE) convolution: a fixed sinusoidal time-positional embedding is added to the trial before temporal filtering, injecting explicit within-trial timing information that plain EEGNet lacks. Everything downstream — the depthwise spatial convolution, separable convolution and linear head — is unchanged, so it is a controlled one-block change to the backbone. Caveat (domain fit): TIE-EEGNet was developed for seizure subtype classification, not motor imagery — its time-positional design targets seizure EEG and may not be well-suited to MI, consistent with it only matching the EEGNet baseline here.
