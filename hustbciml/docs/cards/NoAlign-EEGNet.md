# NoAlign-EEGNet

**Axis:** Alignment &nbsp;·&nbsp; **Paradigm:** MI &nbsp;·&nbsp; **Dataset:** BNCI2014001 (cross-subject LOSO)

**Stage configuration:** `Identity (no alignment) · no-aug · EEGNet · Linear · ERM`

## Mechanism
The no-alignment control: identical to the canonical composition but with the aligner replaced by Identity, so EEGNet sees raw per-subject trials. It isolates the contribution of Euclidean Alignment, and is the baseline for the Channel-Reflection augmentation table because that transform is an electrode-space operation and must run before any spatial whitening.
