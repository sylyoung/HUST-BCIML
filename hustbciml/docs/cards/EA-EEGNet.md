# EA-EEGNet

**Axis:** Canonical reference &nbsp;·&nbsp; **Paradigm:** MI &nbsp;·&nbsp; **Dataset:** BNCI2014001 (cross-subject LOSO)

**Stage configuration:** `EA · no-aug · EEGNet · Linear · ERM`

## Mechanism
Euclidean Alignment recenters each subject's trials by the inverse square root of their mean spatial covariance, so every subject shares a common whitened reference frame; EEGNet — a compact depthwise-separable temporal then spatial CNN — classifies the aligned trials with a linear head under ordinary empirical risk minimization. This is the canonical composition, the reference every controlled single-axis delta in the benchmark is measured against.
