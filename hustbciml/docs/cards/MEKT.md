# MEKT

**Axis:** Classical (network-free) &nbsp;·&nbsp; **Paradigm:** MI &nbsp;·&nbsp; **Dataset:** BNCI2014001 (cross-subject LOSO)

**Stage configuration:** `own CA · tangent-space features · (no backbone) · shrinkage-LDA · fit`

## Mechanism
Manifold Embedded Knowledge Transfer is a network-free classical transfer method. Each subject's trial covariances are centroid-aligned (CA) and mapped to Riemannian tangent-space feature vectors (source subjects pooled). A low-dimensional domain-adaptation projection is then learned jointly for source and target — minimizing the joint-probability distribution shift while preserving source discriminability and target locality, coupled so the two projections stay close — solved as a generalized eigenproblem with EM pseudo-label refinement. A shrinkage-LDA on the projected features classifies the target. Deterministic across seeds.

## Provenance
- **Paper / source:** Zhang & Wu, IEEE Trans. Neural Systems and Rehabilitation Engineering 2020 (Manifold Embedded Knowledge Transfer, MEKT), doi 10.1109/TNSRE.2020.2985996
- **Implementation:** Adapted from Zhang & Wu 2020 (IEEE TNSRE) onto the fit-mode Strategy ABC: pyriemann for the covariance and tangent map, scikit-learn shrinkage-LDA, and the Sec-III-C projection ported from the authors' code (chamwen/MEKT) via TBC-TJU/MetaBCI's mekt_kernel (MIT), with per-subject source CA. MEKT does its own centroid alignment, so the preset aligner is Identity; target features are only ever unlabeled (pseudo-labels come from the classifier, never target labels). Projection hyperparameters (subspace dim, alpha/beta/rho, iterations) default to the MetaBCI reference and are overridable via ctx.cfg.hp. Requires pyriemann + scikit-learn + scipy.

## Note
classical (network-free) transfer, lab method. FULL MEKT: per-subject centroid alignment (CA) + Riemannian tangent-space features + the Sec-III-C domain-adaptation projection (joint-probability MMD + source LDA-scatter + target-locality graph + B~A coupling, solved by a generalized eigendecomposition with EM pseudo-label refinement) + shrinkage-LDA; deterministic (std 0.00). The projection is ported from the authors' code (chamwen/MEKT) via TBC-TJU/MetaBCI's mekt_kernel; it lifts this dataset +6.25 over the CA-tangent core (BNCI2014002 +4.86, BNCI2015001 -0.21). An earlier reading omitted the projection after a buggy attempt collapsed it (~58%); the faithful port improves. No target-label leakage (target features unlabeled; pseudo-labels from the classifier). Lands +4.47 above the EA-EEGNet reference here. Its own CA means the preset aligner is Identity. Requires pyriemann + scikit-learn + scipy.
