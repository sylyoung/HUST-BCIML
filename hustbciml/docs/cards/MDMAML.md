# MDMAML

**Axis:** Transfer / adaptation strategy &nbsp;·&nbsp; **Paradigm:** MI &nbsp;·&nbsp; **Dataset:** BNCI2014001 (cross-subject LOSO)

**Stage configuration:** `EA · no-aug · EEGNet · Linear · MDMAML`

## Mechanism
MDMAML meta-learns an EEGNet initialization across the source subjects with domain-paired first-order model-agnostic meta-learning (MAML): each episode pairs source domains (M/2 pairs), takes one inner adaptation step on one domain and a meta-update from the other, so the learned weights transfer quickly across subjects. At test time the meta-trained network is applied to the target directly, with no target fine-tuning (forward-only) — a privacy-preserving transfer that never touches target data during training.

## Result — 3 seeds (1, 2, 3)

| Metric | Value |
|---|--:|
| Accuracy (mean ± std across seeds) | 75.13 ± 0.38 |
| Δ vs baseline (EA-EEGNet, 72.07) | +3.06 |
| Reference range (expected band) | 70–80 |

Accuracy is the mean over seeds; ± is the standard deviation *across* seeds (a reproducibility figure, not the cross-subject spread). The reference range is the published / expected band on this dataset.

## Provenance
- **Paper / source:** Li et al., IEEE Computational Intelligence Magazine 2022 (meta-learning for fast, privacy-preserving source knowledge transfer), doi 10.1109/MCI.2022.3199622
- **Implementation:** Adapted from Li et al. 2022 (IEEE Computational Intelligence Magazine) onto the Strategy ABC. First-order MAML with an Adam meta-optimizer for the outer loop (a plain-SGD outer loop under-trained EEGNet to near chance; the Adam meta-optimizer is the fix). Many meta-updates per epoch (steps = mean-domain-length // batch). No third-party code beyond the ported meta-loop.
