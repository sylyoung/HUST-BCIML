# ABAT

**Axis:** Transfer / adaptation strategy &nbsp;·&nbsp; **Paradigm:** MI &nbsp;·&nbsp; **Dataset:** BNCI2014001 (cross-subject LOSO)

**Stage configuration:** `EA · no-aug · EEGNet · Linear · ABAT`

## Mechanism
Alignment-Based Adversarial Training, after a clean warmup, perturbs each batch with channel-standard-deviation-scaled projected gradient descent (PGD) and trains on the adversarial examples, hardening the decision boundary over the EA-aligned inputs. The only change from ERM is the adversarial inner loop, which keeps the strategy-axis comparison controlled.
