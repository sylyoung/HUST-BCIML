# CDAN

**Axis:** Transfer / adaptation strategy &nbsp;·&nbsp; **Paradigm:** MI &nbsp;·&nbsp; **Dataset:** BNCI2014001 (cross-subject LOSO)

**Stage configuration:** `EA · no-aug · EEGNet · Linear · CDAN`

## Mechanism
Conditional Domain-Adversarial Network is DANN with the domain discriminator conditioned on the multilinear (outer-product) combination of feature and classifier prediction, so it aligns the joint feature-label distribution across domains rather than only the marginal feature distribution.
