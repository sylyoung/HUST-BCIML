# DJP-MMD

**Axis:** Transfer / adaptation strategy &nbsp;·&nbsp; **Paradigm:** MI &nbsp;·&nbsp; **Dataset:** BNCI2014001 (cross-subject LOSO)

**Stage configuration:** `EA · no-aug · EEGNet · Linear · DJPMMD`

## Mechanism
Discriminative Joint Probability Maximum Mean Discrepancy adds a discrepancy over the joint P(X, Y) — using source labels and target pseudo-labels — that increases between-class transfer distance while decreasing within-class distance, added to the source cross-entropy and trained transductively (the unlabeled target is used during source training). It is the same gradient domain-adaptation family as DAN, JAN and MDD.

Accuracy is the mean over seeds; ± is the standard deviation *across* seeds (a reproducibility figure, not the cross-subject spread). The reference range is the published / expected band on this dataset.

## Provenance
- **Paper / source:** Zhang & Wu, IJCNN 2020 (Discriminative Joint Probability MMD); github.com/chamwen/DaNN_DJP — image-domain paper (Office/Caltech), no EEG benchmark
- **Implementation:** Adapted from github.com/chamwen/DaNN_DJP (Zhang & Wu, IJCNN 2020) onto the transductive Strategy ABC; the paper's DJP-MMD loss (mu = 0.1) is applied to EEGNet features. The paper benchmarks only image domains, so this is a faithful adaptation of the loss to EEG (see the note) — no third-party code beyond the ported loss.

## Note
strategy axis, lab method. Adds the DJP-MMD discrepancy (M_T - mu*M_D over the joint P(X,Y), mu=0.1) to the source cross-entropy over the shared transductive_train skeleton (same family as DAN/JAN/MDD). The paper evaluates DJP-MMD ONLY on image domain adaptation, so there is no published EEG number — the range is a behavior band, not a paper value. On EA + EEGNet it lands +0.13 over the ERM baseline (72.07): a faithful, honest near-baseline result for an image-domain MMD ported to a deep-gradient EEG strategy. Kept as measured. Held-out-source HP selection (lr 1e-3, mu 0.3) lifts it to 73.10 in the benchmark transfer table (2026-07-19 campaign); this repro row keeps the faithful default-config (mu 0.1) number.
