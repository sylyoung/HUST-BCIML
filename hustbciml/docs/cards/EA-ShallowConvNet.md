# EA-ShallowConvNet

**Axis:** Network (backbone) &nbsp;·&nbsp; **Paradigm:** MI &nbsp;·&nbsp; **Dataset:** BNCI2014001 (cross-subject LOSO)

**Stage configuration:** `EA · no-aug · ShallowConvNet · Linear · ERM`

## Mechanism
ShallowConvNet is a single temporal convolution followed by a spatial convolution, a square activation, mean pooling and a log — the FBCSP-style band-power pipeline made differentiable — on EA-aligned trials. Deliberately shallow and tuned for the oscillatory band-power that carries motor imagery.
