# CSDA-EEGNet

**Axis:** Augmentation &nbsp;·&nbsp; **Paradigm:** MI &nbsp;·&nbsp; **Dataset:** BNCI2014001 (cross-subject LOSO)

**Stage configuration:** `EA · CSDA · EEGNet · Linear · ERM`

## Mechanism
Cross-Subject Detail-swap Augmentation applies a db4 discrete wavelet transform to split each EA-aligned trial into approximation and detail bands, then swaps the detail (high-frequency) coefficients between same-class trials from different subjects and reconstructs — synthesizing cross-subject variants that preserve class identity. Because a source batch spans many subjects, the same-class in-batch swap preserves the cross-subject mechanism under the LOSO contract.
