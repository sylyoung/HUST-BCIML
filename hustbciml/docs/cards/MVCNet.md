# MVCNet

**Axis:** Network (backbone) &nbsp;·&nbsp; **Paradigm:** MI &nbsp;·&nbsp; **Dataset:** BNCI2014001 (cross-subject LOSO)

**Stage configuration:** `EA · multi-view aug · IFNet · Linear · MVCNet (CVC + CMC contrastive)`

## Mechanism
Multi-View Contrastive Network is a composite algorithm: it pairs an IFNet backbone (an interactive frequency-domain CNN) with multi-view training — several augmented views per trial supervised by cross-view (CVC) and cross-modal (CMC) supervised-contrastive losses alongside classification. At inference only IFNet and the linear head run. Because it changes more than one stage it is not a single-axis controlled row; it is shown against the reference as a context number.
